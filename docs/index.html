WebSocket 私聊 群组

AI 摘要 智能回复 多模型

🚀 快速开始

几步即可本地运行

完整的前后端项目，按照以下步骤即可快速部署本地环境。

1
克隆项目
从 GitHub 克隆完整仓库到本地
2
配置后端数据库
导入 docs/db 目录下的 SQL 文件，配置 application.yml 中的数据库连接
3
启动 Spring Boot 后端
使用 IDE 或 Maven 命令启动 fendou-chat 模块
4
启动 Vue 前端
进入 fendou-chat-web 目录，安装依赖并启动开发服务器
5
开始使用
访问 http://localhost:5173，注册账号开始体验全功能聊天室
# 克隆仓库 git clone https://github.com/ANnianExplorer/ai-ui-chatRoom.git cd ai-ui-chatRoom # 安装前端依赖 cd fendou-chat-web npm install # 启动前端开发服务器 npm run dev # 后端使用 Maven cd ../fendou-chat mvn spring-boot:run
# 生产构建 npm run build # 预览构建产物 npm run preview
